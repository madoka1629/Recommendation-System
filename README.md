# 基於語意相似度的推薦模型

## 🛠  專案簡介

- 本專案根據使用者所輸入的興趣標籤，於 PTT 論壇的 20 個看板中，推薦使用者可能感興趣的文章。 
- 本方法針對各子板文章標題進行關鍵詞萃取，並計算其與興趣標籤之相似度。 
- 依相似度分數排序，向使用者推薦前三高分的文章。 

## 🛠 使用資料
- 資料集：
  - 撰寫爬蟲程式蒐集 PTT 20 個看板文章內容
- 資料前處理：
  - 正則表達式去除標題中的多餘符號（例如 `[閒聊]` 或 `Re:` 前綴）

## 🛠 使用技術
- 自然語言處理（NLP）：
  - 中文斷詞（jieba）
  - 正則表達式文字清洗
  - 關鍵字提取（KeyBERT + CountVectorizer）
- 詞向量模型：
  - Word2Vec 詞向量模型比對使用者興趣與文章關鍵字語意相似度

## 🛠 評估方法
- 模型依照使用者標籤計算每篇文章的相似度分數
- 選取分數最高的前 15 篇文章作為推薦結果
- 可依實際使用者回饋或點擊率進行效果驗證

1. 本專案預設在 C:\Recommendation System 下執行
2. 下載此壓縮檔並解壓縮，將資料夾命名為 Recommendation System，放置於 C:\
3. 於右側網址下載 PTT 資料集 https://reurl.cc/rKbWNk
4. 於下列網址下載詞向量模型（來源中研院，僅限學術研究，禁止商業用途）
   https://reurl.cc/Xarzle
5. 將(3)、(4)之資料解壓縮後置於 C:\Recommendation System
6. 安裝所需套件：
pip install jieba gensim keybert scikit-learn
7. 執行 main()
